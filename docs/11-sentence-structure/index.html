    第十一章｜句子结构 #

    基本语序 #

    句子 #

    海诺语句子的基本语序是 VSO，即谓-主-宾。例如：

    • séamai dilu.
      “太阳照耀。”
    • al·gabhoi lona craísez.
      “那女孩拿起一本书。”

    在某些情况下，语序会被改变，例如出现次级动词或者需要构造强调句时。介词或者前置词也可能先于动词出现。

    短语 #

    短语的构造遵循一定的语序。如在名词短语中，修饰语通常后置。而在形容词短语中，修饰形容词的副词可以前置或者后置。

    名词短语 #

    名词短语可以分为并列短语，定中短语等。其中并列短语指两个或以上名词同时出现，但并非特指“并列”关系，“而非”，“或”等关系在此书中亦被归在并列短语的范畴下讨论。

    并列短语 #

    表达名词的并列关系时，用连词 o “和”，例如：

    • isi o lóre “小溪和枫叶”
    • harag o thilu “北风和太阳”
    • sór os ecne “羽毛和翅膀”

    当 o 后接塞音时，发生擦化，而后接元音时则变为 os。另外，若出现定指标记，则合成为 osė （单数）和 osar （复数）：

    • ė lona osė davez “那女孩和那男孩”
    • ar mhaiag osar thalve “那些狼和那些渡鸦”

    表达“或”时，用 en ：

    • caffe en laés “咖啡或茶”
    • canna en arma “黄色或紫色”

    除此之外，表达“选择”的含义时还可以用 idci…idci… 或 idci…en… ，译作“要么…要么…”，例如：

    • idci ve idci sana “要么我要么她”
    • idci herace en císace “要么大码要么小码”

    表达“是…而非…”时，用 cón：

    • gichránn cón vohi “面条而不是米饭”
    • hochtogo cón hudar “舞台而非废墟”

    定中短语 #

    一般地，修饰语置于中心语后。修饰语可以是形容词，动词（附属型或分词），副词等。例如：

    • faír ec·tor “坚强的女人”
    • caúis nabha “乌黑的石头”
    • orosse cochad “被杀死的鲸鱼”

    如前文所述，当修饰语描述形状，大小时，修饰语前置：

    • hera i doca “那只大鸟”
    • hecora bólba-hé “那个矮的桶”

    当多个修饰语同时出现时，需要遵循一定的顺序。这个顺序一般是：大小/规模 > 形状 > 颜色 > 材质/性质 > 其他，如：

    • císi biori é phri canni turgi teb “我那些又小又圆又黄又酸的果子”

    短语中存在指示代词的情况下，指示代词永远接在中心语之后，详见名词和代词一章中的指向型代词一节。例如：

    • aesa deis-ma “那个高个子男人”
    • lama-nu gata “远处的青山”

    动词短语 #

    动词短语主要为状中短语和述宾短语。其中述宾短语多用于构词法和一些特定用法。

    状中短语 #

    理论上，任何描述动作状态的副词和动词的组合都可算作状中短语，包括方向前置词，地点前置词或表示数量（如次数）的副词。状语通常置于中心语之前，在最标准的语法中，此时动词都要变为附属型，如：

    • degi ba·damu “我来过两次”
    • ezgai ad·gitte “首次意识到”
    • rabh e·séam “强烈地闪耀”
    • a·mhenve e·tafi “痛苦地学习”

    然而一般用语中，这条规则已经不再严格。另外存在许多状语置于句尾的情况，尤见于 a· 构造的副词。这种情况下，除了非常正式的书面语之外，主动词一般不需要变为附属型。

    • a·mhenve enni·tafii síanihi.
      “那学生正在痛苦地学习。”
    • en·tafii síanihi a·mhenve.
      “那学生正在痛苦地学习。”

    以上两句意思相同，但第一句在语法上更为“正确”。

    述宾短语 #

    述宾短语主要用于构词，从句。在构词时，语序为宾-谓，如：

    • iur + mest- -> iurmesti “观星”
    • fova + mest- -> fómesti “赏花”
    • eri + tach-ú- -> eridachí “作诗”，“写诗”
    • bide + car- -> bideghiar “畅想”（lit. 远望地平线）
    • sora + heg- -> soraheíg “熬夜”（lit. 等待夜晚）

    注意，符合条件时，动词词首发生浊化音变。

    在从句中，谓语动词变成附属型或 ito 动名词。ito 动名词的情况下，短语语序始终为宾-谓：

    • e·cogg in naséog “烤面包”
    • dorbi ialleto “举办祭典”

    形容词和副词短语 #

    形容词和副词短语的语序比较自由，可前可后。需要注意的是，程度副词置于形容词之前时，形容词词首发生擦化：

    • dade chodora “非常复杂”
    • hoso hirna “稍微有点冷”
    • utta lesta “完全干净”

    形容词除了和副词组合，还可以通过加后缀的方式表达程度，详见前文。

    介词短语 #

    介词短语可以分为前置介词短语和后置介词短语。由于介词变位的存在，变位后的介词也算作介词短语。介词短语在句中充当状语或宾语的成分，如：

    • no cub-ma “在这个国家”
    • nos taf “越过沼泽”
    • cengav “东侧”，“东边”

    当介词短语作为宾语出现在另一个短语，如动词短语中时，介词会独立化1：

    • addafí agar patito “询问老师”
    • e·bar anvé ali “走上前线”

    1. 见介词独立一节。 ↩︎
