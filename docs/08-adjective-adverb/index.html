    第八章｜形容词和副词 #

    在这本书中，除前文的冠词和指示代词，一切“修饰”名词的词语都被当作形容词讨论，因此由动词构造的定语也会在本章内出现。

    形容词 #

    形容词的结构 #

    形容词多数以 -a 结尾，少数没有规则的词尾可供判断。某些形容词的词尾可以被归类，如表示性质，材质，可能性，对事物的影响等。以下是一些常见词尾：

    词尾 含义 例词
    -a 无特殊含义 guna “古老的”
    -sa/-ssa 某种性质 thencisa “有问题的”
    -ra 对外部影响 tandira “保暖的”
    -ea/-ia 自身特点 taroia “金色的”
    -da 材质 tarda “金制的”
    -atta/-etta 可能性 dyvetta “可数的”

    有多个形容词形容一个名词，一个形容词同时形容两个并列名词，或名词为定指复数时，形容词需要采用并列形态，即把词尾改为i或添加i：

    • bae achli mizi
      “又黑又窄的路”
    • egúz tardi rutti
      “奇怪的金戒指”
    • lona o dhavez soreri
      “美丽的少女和少年”

    形容词的功能 #

    形容词主要有做定语，做表语和名词化几种功能。做定语时，形容词需要遵循一定的语序。大多数形容词需要位于中心语之后，而描述形状，数量的形容词则需要在中心语之前。

    • é phri canni
      “那些黄色的果子”
    • biordi é phri
      “那些圆的果子”
    • heri aesi caúis
      “又大又高的石头”
    • cá boca
      “很多的树”

    形容词不根据名词本身的单复数变形。

    做表语时有两种说法，一种是表语前置，主语后置，两者间以 ghe 或 dhe 连接（在标准用语中 ghe 更常用）。另一种是 ghe 前置，而表语放在末尾，这种用法常用于描述事实或者常识：

    • osca ghe tono-ma.
      “这苹果是绿的。”
    • ghe tono-ma osca.
      “这苹果是绿的。”
    • ghe maísolu sanza.
      “海水是咸的。”

    如果要强调表语，则可以用 dem 后加表语再加主语的句式。注意，这个句式只能用于定指：

    • dem osca i dono.
      “这苹果是绿的。”

    形容词名词化可分为两种：指人和指物。指人的情况下需要用到环缀 ib-...-i ，而指物需要 to-...-i 环缀：

    • ib-guni “老的（人）”
    • to-guni “老的（东西）”

    比较级和最高级 #

    形容词比较级的构成由置于形容词之前的 itte 构成，此时形容词词尾变为 -i 。被比较的对象会加上后缀 -dad ，或者用一些介词引导，如 ce “比”，ali “上”，ilan “出来”，be “下”等：

    • séginn itte tengisi
      “更尖的小刀”
    • itte ib-aesi civi
      “我们中更高的那个”
    • ghe itte ebi ma voito ali.
      “这个饮料更甜。”
    • ghe’v itte matissi isdad.
      我比你更聪明
    • itte leisi ghe ma barcha ce i·neiv.
      这辆车比那辆慢。

    而最高级则由 egco 引导，有时会在形容词后加 to ，而比较对象可用 ci 或者 nan 引导：

    • dhe i aspé davaloí egco vaósi to.
      “口号是最无力的承诺。”
    • ghe bava-hé egco tissito naniu.
      “那座塔是它们中最低矮的。”
    • dhemi doro egco divri.
      “这是最贵的蛋糕。”
    • ghe i doro egco divri.
      “这蛋糕是最贵的。”

    某些形容词有独立的比较级和最高级，比如 hata “好的”和 fera “坏的”：

    原型 比较级 最高级
    hata eitte eisco
    fera crette cregco

    这些形容词在名词化时，会把尾元音替换成 -i 。

    在表达平级，也就是“x 和 y 一样”的时候，在形容词上使用前缀 bi- ，比较对象主要用 ilan 引导。注意，bi- 会引起浊化，而形容词开头为元音时变为 bin- ：

    • bigísa ilanu
      “和我一样小”
    • iuben dhesgeth binachla ghe calcoth ilan gheala!
      “明明是白天却和夜里一样黑！”

    形容词的极化 #

    形容词表达的情感倾向可分为正极化，负极化和中性。极化主要与形容词本身的语义有关，如：

    • sorera “美丽的”（正极化）
    • cusia “令人失望的”，“糟糕的”（负极化）
    • ára “厚的”（中性）
    • cryra “恐怖的”（负极化）

    然而，在某些语境下，这种倾向有可能发生变化。尤其是在某些群体的口语中，经常使用负极化的词表达正向倾向：

    • dem cryró i merúgal!
      “那电影屌爆了！”
      （lit. “那电影太恐怖了”）
    • erbitta ghe seúb tis.
      “你的衣服真好看。”
      （lit. “你的衣服真破。”）
    • nirutava!
      “精彩绝伦！”
      （lit. “骇人听闻”）

    这种用法通常只在小圈子内流通，但也有逐渐进入主流口语的用法，例如上述使用 cryra “恐怖”表示“赞”，“牛逼”，“屌爆”和 nirutava “骇人听闻”表示“精彩”的例子。

    另外，被修饰词也会影响到形容词的倾向，而这部分与海诺语中一类特殊的词汇，以及海诺文化中对事物本身携带“情感”的解释有关。详细内容可参见后文[环境词]。

    程度 #

    要表达形容词的程度，有两类方法：

    1. 使用副词
    2. 变化形容词的形态

    程度副词可放在形容词前，也可放在形容词后，如：

    • coghea dade “非常高兴”
    • dade choghea “非常高兴”

    此处当表示程度高的副词 ore 置于形容词前时，形容词发生了词首辅音弱化，这是因为在 ore 的古早形式中，其词尾有一个 -s ，而在现代逐渐脱落所致。这类副词一般由别的词类派生而来，详细可参考后文派生副词。

    第二类方法是变化形容词的形态，多数是在词尾加后缀。以下是一些表示程度的后缀：

    后缀 含义 例词
    -ó/-obh 超，非常 rabió “超级有钱”
    -sig 太，过分 ebisig “太甜了”
    -ppar 完全不，一点也不 sirnappar “完全不冷”
    -ora 非常，十分 ornora “非常温柔”
    -sínn 有点，稍微 sanzasínn “有点咸”
    -gire 足够，满足 hatagire “足够好”

    后缀还可以和程度副词或前置词搭配表达更细腻的含义：

    • cun aphassig “有点太搞笑了”
    • coran·daderó “超级危险”
    • las denzegire “有够疯狂的”

    形容词的构成 #

    如前文所述，有一部分形容词，或者带有形容词性质的词由动词构造。构造方法是将动词变为附属型，有时会将 e- 替换为 a- ：

    • ratta ad·ionte
      “可爱的猫”（d·ionte “使人心动”，“萌”）
    • domu enni·peg
      “正在爬的虫子”（peg- “爬行”）
    • Soscoisi a·tamire
      “即将到来的圣诞节”（tam- “来”）

    另外，诸如 sa-, -sce, -mu等词缀还可以从名词构造形容词，通常带有超越字面的含义。这些词缀的字面含义为“像”，“好似”。其中，-mu 只能跟在有生命的名词后：

    • sa- + maer -> samaer “如银一般的”（意为“高贵的”）
    • caúis + -sce -> caúisce “像石头一样的”（意为“顽固的”，或者“坚定的”）
    • fova + -mu -> fovamu “花似的”（意为“美丽的”，“动人的”，或“残酷的”）

    这类“形容词”不同于一般的形容词，没有准确的含义。它们的含义需要通过上下文判断，经常出现在诗歌和文学性比较强的文章，或者戏剧，影视台词中。

    副词 #

    副词的功能是修饰动词，形容词，或者表示状态，做状语，做补语等。副词没有形态变化，可以由其他词类派生而来，也有原生的副词。除了派生副词，其他副词无法通过形态判断。

    副词在句子中的位置比较自由，但通常位于动词前，或者句尾，取决于是否需要强调。

    派生副词 #

    从其他词类派生副词一般有四种方法，第一种是加前缀 a· ，这种方法适用于名词或者动词。第二种只适用于形容词，方法是将形容词词尾的 -a 变成 -e 。第三种适用于动词，如前文说明的伴随动作。而第四种则是下文的有关地点方位和时间副词的构成方法。

    一些例子如下：

    • a· + curu -> a·churu “按顺序”，“依次”
    • a· + gir -> a·gir “在空中”，“徒劳地”
    • fassa -> fasse “充分地”，“够”，“满”
    • edisa -> edise “明亮地”，“变亮”
    • ess- + -en -> essen “哭着”
    • ciz- + en -> cizden “吃着”

    地点方位副词 #

    表示地点或方位的副词可以通过方向前置词加方位名词构成，如：

    • no + cen -> nocen “在东边”
    • be + hecc -> behecc “在南边”
    • im + sél -> imsél “在西边”
    • al + cur -> alcur “在北边”

    在海诺传统方位观中，不同方位需要用不同的方向词引导，但现代标准语里可以统一为 no 。

    若要表示来自和朝向某个方位，则用 -car 和 -aí ：

    • cen + -aí -> cenaí “向东”
    • sél + -car -> sélcar “自西”
    • cur + -car -> curcar “自北”
    • hecc + -aí -> heccaí “向南”

    如果将传统方位观里的方向前置词反过来，也可以表达来自那个方位的意思。

    • al + hecc > alhecc “从南方来”
    • be + cur > becur “从北方来”

    前后左右分别是 men, aén, ach 和 var 。这些词可以像方向前置词那样独立使用，比如：

    • men ramha ve.
      “我往前走。”
    • ach os·dégu!
      “往左转！”

    另外，方向也可以对应前后左右，它们的对应关系是：东-前，西-后，北-左，南-右。

    时间副词 #

    表示之前和之后的时间副词分别是 men（与空间的“前”相同！）和 síb 。表示空间的 al 和 be 加上 sog “沿着”也能表示时间的前后：alsog “之前”，besog “之后”。

    表示日子的时间副词有：

    词语 含义
    fogin 昨天
    tim, iam 今天
    erach, síub 明天
    imfogin 前天
    cenerach 后天
    timala, sagheala, magon 今晚，今夜
    fogala, fogon 昨晚，昨夜
    erala, eragon 明晚

    这些副词一般放在句首或句末，而放句首时它们可以附带人称附着词，对应施事者，并将句子里的主动词变为附属型：

    • fogala conda ve i vosag.
      “昨晚我参加了宴会。”
    • fogala’v e·cond i vosag.
      “昨晚我参加了宴会。”
    • cenerachis ero·cha i derg?
      “你明天要去镇子上吗？”

    如果要将时间地点副词变为限定形式，即做定语，则将其置于中心语后并用i连接。

    • deis i alhecc
      “来自南方的男人”
    • coth i síb
      “后面的事”，“后来的事”
    • i gathevís i cenaí
      “那辆开往东方的列车”

    程度副词 #

    程度副词表示形容词的程度，一些常见的程度副词有：

    副词 含义 极化
    dade 非常，十分 中性
    cun 有点，稍微 中性
    utta 完全，甚，极其 正极化
    las 完全，极度 负极化
    eleci 更 中性
    hoso 稍微，一丁点 正极化
    dían 有点，稍微 中性 / 负极化
    atav 适当，合适 中性
    córna 超级 正极化
    zangi 超级 负极化

    如前文所述，程度副词可以放在形容词前或后，放在前面时形容词发生词首辅音弱化。

    与形容词类似，程度副词也有极化的区别。如上表所示，对于不同极化的形容词，能够进行修饰的副词也不同。另外，情感倾向发生变化的现象也存在于副词中，如使用 zangi “超级（负极化）”来表达正向情绪：

    • zangi horro?
      “太棒了吧？”
