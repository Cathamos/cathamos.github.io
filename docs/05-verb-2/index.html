    第五章｜动词基本结构 #

    简单动词 #

    概述 #

    如前文所述，简单动词即为动词本体和后缀就能够表达含义的动词。最简单的判断方法是动词中没有中点 · 分隔，如：cara “看”，sése “睡”。cara 可以拆分为car-a，而 sése 可以拆分为 sés-e。

    目前尚不清楚简单动词和动词本身含义是否简单有没有直接关系，根据最近的调查显示，某些简单动词的有着比较复杂的意思，如 ethona “怀疑”。推测原因是这些动词在古老的时代仍是复合动词，但随着时代更迭逐渐固定为简单动词。亦存在一定数量的外语借词，以简单动词的形式出现而包含比较复杂的含义，如 tígure “稳定”，ciráse “掌握”，“了解”，这类词一般由三个辅音组成，第一个元音多为i。而这样的借词也有被构造为复合动词的情况，如 ní·cirás “解释”。

    简单动词添加人称的方式 #

    海诺语的配列系统呈现出一部分活格（fluid-S）特征，因此简单动词添加人称有两种方式：

    1. 添加独立代词；
    2. 添加人称附着词。

    及物动词的情况，使用独立代词：

    • posa sana todd “她听声音”
    • voi ve galo “我喝酒”

    不及物动词的情况，使用人称附着词或独立代词，使用人称附着词时往往带有不自觉的意味：

    • tama sana “她来”
    • sése isi “你们入睡”
    • sése asu “你们强行睡觉”

    在口语中，ev 可能变为 o ，asi/si 可能变为 isí/sí ，如：

    • men a dhén isí “你们在干嘛”
    • coghea o “我乐意”

    当动词为及物动词时，也有可以使用人称附着词的情况。此时句子暗示一种不自觉的行为，或者受到某种影响而进行动作。试比较：

    1. posa sana todd “她听声音”
    2. posa ana (in) todd “她听到声音”

    第一句可能指她主动去听声音，而第二句暗示她不经意听到了声音。此时宾语可以带上一个 in 的标记。不自觉行为除了这种方式，还可以通过构造复合动词来表达：dan·phos todd “她听到声音”。

    构造类似“x 到”的动词的前置词是du·。

    复合动词 #

    概述 #

    复合动词顾名思义，就是由前置词和动词本体复合而成的动词。在海诺语中，动词以复合动词的形式存在的情况较多，因此复合动词的掌握是学习海诺语法的重中之重。

    一个复合动词的基本结构是：

    前置词1-人称附着词-[前置词2-...-前置词n]-（前置词0）-动词附属型-[后缀1-后缀2-...-后缀n]

    虽然前置词可以相互叠加，但在现代海诺语中，一般只能叠加三个前置词。人称附着词，即中缀一般添加在第一个前置词后，且并不强制要求。中缀也会和之前的前置部分缩合，并引起其后部分发生辅音交替。前置词的叠加遵循一定顺序，一般按照 语气 >方向（空间）> 时间（体貌/时态）> 其他 的顺序来叠加。

    前置词0 指的是，在某些本身就是复合动词的动词的情况下，原本的前置词与动词结合紧密，导致在其前方加入前置部分时，这个原本的前置词会与其后的动词缩合；又或者是某些特定的前置词，如 is· “导致”，会强制使其后的前置词与动词融合。一个典型的例子是 ci·attí “捡到”：

    原动词 is·
    ‘捡到’ ci·attí is·cettí

    此处的 ci·attí 就被 is· 强制缩成了 ·cettí 。这种用法通常用于书面语，而在现代口语中非常罕见，被另一些表示“导致”，“结果”的词，如 phencais，cais 替代。

    前置词 #

    前置词可被细分为语法性前置词和构词性前置词。

    语法性前置词承担着助动词的功能，即表达动作的状态，方向，倾向等含义。在海诺语中，这些表达状态的前置词通常会被看作动词主要部分，而表达动作本身含义的词根则“退化”为附属型——一种类似副词的成分。因此，语法性前置词通常与动词的体貌，时态，语气，方向等要素挂钩。它们之间尤其是表示方向和时间的前置词可以互相叠加，如前文例子：

    • ileren·asir i lama anu.
      “他从那极远的山一直走着来”

    而构词性前置词与简单动词结合构造出复合动词，这些复合动词之上可以叠加语法性前置词，多数会与构词性前置词结合。狭义地说，只有构词性前置词和简单动词结合的产物能被称为“复合动词”，但为了方便讨论，本书将所有带前置成分的动词都视为“复合动词”。

    复合动词中的人称附着词 #

    复合动词中的人称附着词通常添加在前置词与动词词根之间，这并不是强制的：在实际的海诺语中时常观察到不带人称信息的动词。（而且不是所有前置词都能添加人称附着词！）注意，这种情况下，人称信息可能表示施事者，根据前置词的含义也可能表示受事者，或者某些中动态的主语。若要明确表示施事者，则可以通过在动词后添加独立型人称代词实现。

    人称附着词会与前置词发生缩合，并且有可能引起其后动词词首的辅音交替现象，例如：

    • atta· + <s> + ·torra -> atta·thorra “你能够忍受”
    • ro· + <v> + ·piri -> ro·biri “我将铭记”
    • ci· + <ir> + ·attí -> cer·attí “捡到他”
    • cá· + <vi> + ·sta -> cói·sta “让我们游行”

    有关常用前置词与人称附着词缩合的形式，以及附加的人称信息表示施事还是受事，参考前置词详解。

    后缀 #

    复合动词和简单动词都可添加后缀。复合动词通过前置词与后缀的配合来表达更为精细的意思，以上面的两个动词为例：

    1. ro·biri + -in -> ro·birgin “我想着铭记，而且一定会铭记”
    2. cói·sta + -u -> cói·staíu “让我们游行吧”

    1 中的 -in 后缀表示确定，肯定的语气，或者是表达决心。2 中的 -u 则表示强调的口吻，有时含有命令的意思。

    另外有一些体貌要求前置词与固定的后缀搭配，比如经历体就需要通过 eb·VERB-u 构造。

    一些常见的后缀有：

    • -ó，加强语气，通常与 cá· 劝告语气搭配使用
    • -ur，描述义务，必须，应该做
    • -u，表示强调，命令等，也常与劝告语气搭配
    • -in，应该要做，确定，肯定，决心
    • -ion，表示目的，为了做
    • -ire，将行体，正要开始做
    • -en，持续体，处于某种状态
    • -é，提高动词地位，使其变为主要动词，或提高生命度，或作为前文回指
    • -ad，表被动，通常只用于被动分词的构造

    绝大部分添加在动词上的后缀都是元音开头，因此添加后缀时可能会插入所谓的介音。介音的类型与动词词跟的类型有关，如 cói·sta 是元音词根，sta-í-，因此添加后缀时变为 cói·staíu ，而 ro·biri 是辅音中缀词根，pir-g-，因此添加后缀时变为 ro·birgin 。这些不同词根类型的特征会在动词的独立型上体现出来，具体分类请参考下文动词词根的类型。

    如前文所述规范，定指标记 ė 和 ar 有可能会附着在动词之后。例如：

    • aé to·cacazar dhuísti.
      “敌人已经发觉了。”
    • nograi chún.
      “那龙虾逃走了。”

    在动词结尾是元音的时候，ar 会简化为 -r 。

    • ingéc·luchur phoscozo!
      “不准跟那些流氓鬼混！”
